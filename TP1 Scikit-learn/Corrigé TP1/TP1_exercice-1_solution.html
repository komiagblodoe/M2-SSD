  • Pour raliser une rgression polynomiale on s'appuiera sur la classe PolynomialFeatures du module preprocessing de Scikit-Learn.
  • La procdure consiste :
    1. transformer les donnes en utilisant la fonction fit_transform de l'objet PolynomialFeatures, ce qui a pour effet de raliser l'expansion polynomiale.
    2. apprendre un modle de rgression linaire, en utilisant la classe LinearRegression, partir de la martrice obtenue (et du vecteur $y$).
  • A noter que pour la prdiction, il faut galement transformer les donnes avant d'appeler la mrthode predict de l'objet LinearRegression.

# define list of degrees to consider
d_list = [1,2,3,5,9]

# show points
plt.scatter(x,y)

for i in range(len(d_list)):
    polyTransform = PolynomialFeatures(degree = d_list[i])
    x_poly =  polyTransform.fit_transform(x)
    linReg.fit(x_poly,y)
    pred = linReg.predict(x_poly) 
    x_grid_poly =  polyTransform.fit_transform(x_grid)
    pred_grid = linReg.predict(x_grid_poly)
    rmse_train = np.sqrt(np.mean( (y-pred)**2  ))
